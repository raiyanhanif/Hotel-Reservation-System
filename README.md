JDBC CRUD Application (Java + MySQL)

This project is a menu-driven Java application developed using JDBC to perform complete CRUD operations on a MySQL database.

Features

Establishes database connection using JDBC

Insert, retrieve, update, and delete records

Uses PreparedStatement for secure SQL execution

Handles user input via Scanner

Exception handling for database operations

Technologies Used

Java

JDBC

MySQL

Learning Outcomes

Strong understanding of JDBC architecture

Hands-on experience with database connectivity



Learned difference between Statement and PreparedStatement

Practical experience with SQL and Java integration
